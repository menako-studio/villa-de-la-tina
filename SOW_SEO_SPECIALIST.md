## 1. PROJECT OVERVIEW & SEO OBJECTIVES

### 1.1 Background
Villa De Latina Kuningan requires comprehensive SEO strategy and implementation to ensure strong organic visibility in search engines, particularly targeting constituents in Kuningan, Ciamis, Banjar, and Pangandaran regions. The website serves as a digital portfolio for Hj. Tina Wiryawati's community initiatives and political involvement.

### 1.2 SEO Objectives
- **Primary Goal**: Establish strong local search presence in target regions (Kuningan, Ciamis, Banjar, Pangandaran)
- **Secondary Goal**: Optimize for community event and political portfolio related searches
- **Technical Goal**: Ensure website is properly indexed and crawlable by search engines
- **Content Goal**: Optimize all pages for relevant keywords while maintaining natural readability
- **Performance Goal**: Lay foundation for organic traffic growth post-launch

### 1.3 Target SEO Metrics
- Website indexed in Google Search Console within 2 weeks of launch
- All 5 pages optimized with unique metadata
- Minimum 10 target keywords identified and integrated
- XML sitemap created and submitted
- Basic local SEO setup completed
- Google Analytics and Tag Manager properly configured
- Technical SEO issues (if any) documented and resolved

### 3.2 XML Sitemap Creation & Submission

**Sitemap Structure**:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://villadelatinakuningan.com/</loc>
    <lastmod>2025-12-01</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://villadelatinakuningan.com/tentang-kami</loc>
    <lastmod>2025-12-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <!-- Additional pages... -->
</urlset>
```

**Specifications**:
- Include all 5 main pages
- Set appropriate priority levels
- Define changefreq based on content update patterns
- Ensure clean URLs without parameters

### 4.2 Header Tag Structure (H1, H2, H3)

**Best Practices**:
- One H1 per page (main page title with primary keyword)
- H2 for major sections (include secondary keywords)
- H3 for subsections
- Logical hierarchy without skipping levels

**Example Structure - Homepage**:
```
H1: Villa De Latina Kuningan - Pusat Event & Acara Komunitas
  H2: Acara Mendatang
    H3: KOPIKU Soft Launching 2026
    H3: Trail Run Kuningan 2026
  H2: Berita Terbaru
  H2: Tentang Kami
```

### 4.3 Image Alt Text Optimization

**Alt Text Guidelines**:
- Descriptive and concise (10-15 words max)
- Include keywords naturally when relevant
- Describe what's in the image
- Don't start with "image of" or "picture of"

**Examples**:
```
Bad: "img001.jpg"
Good: "Hj Tina Wiryawati bersama masyarakat Kuningan di acara Wawasan Kebangsaan"

Bad: "event.png"
Good: "Suasana Trail Run Kuningan 2026 di Villa De Latina"

Bad: "photo"
Good: "Villa De Latina Kuningan tampak depan dengan taman hijau"
```

**Deliverable**: Image alt text recommendations for all website images

## 5. CONTENT OPTIMIZATION

### 5.1 Keyword Integration in Existing Copy

**Optimization Process**:
1. **Review Draft Content**: Analyze existing copywriting
2. **Identify Integration Opportunities**: Find natural places for keywords
3. **Rewrite for SEO**: Optimize without sacrificing readability
4. **Maintain Natural Flow**: Avoid keyword stuffing

**Keyword Density Guidelines**:
- Primary keyword: 1-2% density
- Secondary keywords: 0.5-1% density

### 5.3 SEO-Friendly Content Guidelines for Future Updates

**Content Creation Best Practices**:

**Writing Guidelines**:
1. **Keyword Research First**: Identify target keyword before writing
2. **Compelling Headlines**: Use power words and keywords
3. **Strong Introduction**: Hook readers and include keyword in first 100 words
4. **Scannable Content**: Use short paragraphs, bullet points, subheadings
5. **Value-Driven**: Answer questions, provide solutions
6. **Call-to-Action**: End with clear next step

**Technical Guidelines**:
1. **Word Count**: Aim for 500+ words per page/post
2. **Keyword Placement**: Title, H1, first paragraph, URL, meta description
3. **LSI Keywords**: Use related terms and synonyms
4. **Multimedia**: Include images, videos when relevant
5. **Mobile-Friendly**: Ensure content displays well on mobile

**Content Maintenance**:
- Update content every 6-12 months
- Refresh old blog posts with new information
- Add new internal links to updated pages
- Monitor performance and optimize underperforming pages
